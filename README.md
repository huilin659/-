## 💡 使用指南

1.  **获取资源**：
    - 点击上方的【夸克网盘总链接】。
    - 进入后，**使用页面的搜索框**，输入你想要的**游戏名称**进行精准查找。
    - 找到后，直接点击转存或下载。

2.  **寻求帮助**：
    - 如果链接失效或找不到你想要的游戏，请前往 [**【Issues反馈区】**](https://github.com/huilin659/Awesome-Game-Resources/issues) 发帖说明，我会尽快处理。

3.  **关注更新**：
    - 点击仓库右上角的 `Watch` 按钮，选择 `Releases only`，即可在我批量更新游戏列表时收到通知。
